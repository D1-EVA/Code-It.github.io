Given an integer A.Write binary representation of the integer without leading zeros.Flip all bits then return the integer value of the binary number formed.Flipping means 0 -> 1 and 1 -> 0.

Constraints: 

1 <= A <= 109

Input Format: 

Given an integer A.

Output Format: 

Return an integer.

Example Input: 

Input 1:
A = 7

Input 2:
A = 5


Example Output: 

Output 1:
0

Output 2:
2


Example Explanation: 

Explanation 1:
7 -> 111 -> 000 ->0
Explanation 2:
5 -> 101 -> 010 ->3
