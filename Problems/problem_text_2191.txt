Given an integer A, check if it is an Armstrong number of not.An Armstrong number is a number that is the sum of its own digits each raised to the power of the number of digits.

Constraints: 

1 <= A <= 109

Input Format: 

Given an integer A.

Output Format: 

Return an integer.

Example Input: 

Input 1:
A = 371

Input 2:
A = 123


Example Output: 

Output 1:
1
Output 2:
0

Example Explanation: 

Explanation 1:
333 + 777 + 111 = 371
Explanation 2:
111 + 222 + 333 != 123
